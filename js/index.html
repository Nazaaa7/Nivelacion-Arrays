Sistema de Gestión de Personas

Instrucciones de Uso

Para utilizar la aplicación JavaScript:

  1. Abra la consola del navegador para interactuar con el programa:
    • Windows/Linux: Presione F12 o haga clic derecho → Inspeccionar → Consola
    • Mac: Presione Cmd+Option+J o haga clic derecho → Inspeccionar → Consola
  2. Una vez en la consola, el programa ya estará ejecutándose
  3. Siga las instrucciones que aparecen en la consola para utilizar el programa

Nota: Este programa funciona mediante la consola del navegador. Si no ve ninguna interfaz gráfica en esta página, es normal y esperado. La interacción ocurre completamente en la consola.

Si desea volver a iniciar el programa en cualquier momento, puede ejecutar manualmente la siguiente instrucción en la consola:

gestionPersonas();

Funcionalidades

  • Agregar personas (nombre, edad, nota)
  • Visualizar listado original de personas
  • Visualizar listado ordenado por nota (mayor a menor)
  • Validación de datos de entrada
